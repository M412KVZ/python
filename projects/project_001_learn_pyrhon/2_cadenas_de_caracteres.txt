# En programacion las cadenas de caracteres o mejor conocidas como strings(str),
es una serie de caracteres compuestas por letras, numeros, signos y simbolos, que dentro de sus funciones, destaca la interaccion de un programa con el usuario.

# En Python existen distintas operaciones para manipular una cadena de caracteres (str), dentro de las cuales se encuentran;

	la asignacion
	la concatenacion
	la busqueda
	la extracccion
	la comparacion

# La asignacion, consiste en asignar una cadena de caracteres a otra, para lo cual es necesario utilizar el operador "+=".

	mensaje = "hola"
	mensaje += " "
	mensaje += "Ernesto"
	print(mensaje)
	resultado = Hola Ernesto

# La concatenacion es una operacion que consiste en unir dos cadenas o mas, para formar una cadena de mayor tamaño. Para lo cual es necesario utilizar el operador  +.
	mensaje_1 = "Hola"
	mensaje_2 = " "
	mensaje_3 = "Carrera"
	mensaje_4 = "Ernesto"
	print(mensaje_1 + mensaje_2 + mensaje_4 + mensaje_2 + mensaje_3)
	resultado = Hola Ernesto Carrera

# Concatenacion de texto y numeros, para esto se debe convertir el mumero a str y duspues concatenarlo.

	numero_1 = 2
	numero_2 = 4
	resultado = numero_1 + numero_2
	resultado = str(resultado)
	print("El resultado de la suma es: " + resultado)

# La busqueda, consiste en localizar dontro de una cadena, una subcadena mas pequeña a un caracter, para lo cual es necesario utilizar el metodo find.

# La extraccion, se trata de sacar fuera de una cadena una porcion de la misma segun su posicion dentro de ella.

para ello es necesario indicar la posicion a extraer [1:8]

	mensaje = "Hola Ernesto"
	extraer_subcadena = mensaje[5:12]
	print(extraer_subcadena)
	resultado = Ernesto
